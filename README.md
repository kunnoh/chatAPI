# charRESTapi nodeJS nestJS framework
## Clone the Application from github
```
git clone https://github.com/kunnoh/chatRESTapi.git
```
## Installation
Make sure you have installed nestjs
```
npm install -g @nestjs/cli
```

install application dependencies from npm for the application
```bash
npm install
```

## Running the app
<ul>
  <li>
    development
    ```bash
    npm run start
    ```
  </li>

  <li>
    run the app and debug
    ```bash
    npm run debug
    ```
  </li>

  <li>
    lint the code
    ```bash
    npm run lint
    ```
  </li>

  <li>
    format the code using prettier
    ```bash
    npm run format
    ```
  </li>

  <li>
    production mode
    ```bash
    npm run prod
    ```
  </li>
</ul>


## Test the app
unit tests
```bash
npm run test
```
e2e tests
```
npm run test:e2e
```

test coverage
```
npm run test:cov
```
